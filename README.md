# Paragon Outbreak

## Overview

Paragon Outbreak is a fast-paced 2.5D action sci-fi arcade shooter inspired by classic games like DOOM and Wolfenstein.
The game uses a **raycasting technique** to simulate a 3D environment while maintaining a retro visual style.

## Story
The player suddenly wakes up inside a strange version of Paragon University, as if trapped in a nightmare. The environment feels distorted and unfamiliar—hallways twist into a maze, and something is clearly wrong. Zombies roam the campus, turning the once normal university into a dangerous labyrinth. With no clear explanation of how he got there, the player must fight through the maze, survive waves of enemies, and find a way to escape.

Is this real… or just a dream?

---

## Features

* Retro-style raycast FPS gameplay
* Multiple weapons (gun, machine gun, mini gun, knife)
* Zombie enemy system
* Health and damage system
* Game over + restart system
* Progressive difficulty
* Final escape objective (ending)
* Bonus / Easter egg content

---

## Controls

* Move: **W A S D**
* Shoot / Attack: **Space**

* `scene/` → game scenes
* `script/` → gameplay logic (player, enemies, UI)
* `assets/` → textures and models
* `animation/` → animations
* `sound/` → audio files
* `pictures/` → images/screenshots
