# Paragon-Outbreak

**Paragon Outbreak** is a fast-paced **2.5D action sci-fi casual arcade** shooter inspired by **DOOM / Wolfenstein**, using a **raycasting** method to create a **3D-like environment** while keeping a retro look.

A virus outbreak strikes **Paragon University**, turning students and staff into zombies. You play as a surviving student trapped on campus—fight to escape while uncovering the cause of the outbreak.

## Core Gameplay
**Start → Finish**
1. **Start / Tutorial**
   - Spawn inside campus
   - Learn: movement, shooting, picking up items
2. **Mid Game**
   - Clear designated rooms
   - Find **resources** and **keys**
   - Zombies increase in number and pressure
3. **End Game**
   - Reach the escape point
   - Final zombie encounter
   - View stats/results

## Controls
- **Move:** `W A S D`
- **Shoot:** `Space`
